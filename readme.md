## Running the App

From the root of the app run: 

```baash
  $ cp .env.example .env # add your database credentials
  $ composer install
  $ php artisan migrate
  $ php artisan db:seed # the database gets populated through the API
  $ php artisan key:generate
```

If for some reason the front end is acting up, please run: 

```bash
  $ npm run development
```
